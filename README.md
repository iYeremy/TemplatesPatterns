# Design Patterns
Repositorio con plantillas base para patrones de diseño.

## Patrones Incluidos
### Strategy: Intercambio de algoritmos en tiempo de ejecución.

### Bridge: Desvincula la abstracción de su implementación.

### Singleton: Instancia única y global de una clase.

### Flyweight: Ahorro de memoria mediante objetos compartidos.

### Chain of Responsibility: Cadena de objetos que procesan una solicitud.

### Repository: Capa intermedia para el acceso a datos.

### Dependency Injection: Inyección externa de dependencias.

### Observer: Notificación automática de cambios de estado.

### Builder: Construcción paso a paso de objetos complejos.
